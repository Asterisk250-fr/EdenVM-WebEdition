# EdenVM (web friendly edition)
A virtual machine emulator for web and download. Use your own ISO or the already listed ones.

# Features

-HDD/SSD/NVME/VHD/QCOW2 Disk creation, does not take up real space!

-Pick a CPU. Pick any x32, x64 and ARM CPU.

-Pick a GPU. Pick any gpu and choose video memory

-Operating system emulation, boots any iso you give it! comes with preinstalled linux ISO's

-Boot multiple systems at a time

-Change device type
